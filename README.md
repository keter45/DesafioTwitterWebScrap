# README

Exemple:

https://media.giphy.com/media/RgnFCKJNZGIwBc6jj8/giphy.gif

Ruby version:

    $ ruby '2.5.3'

To run this project execute:

    $ bundle install

And to create and generate Users table:

    $ rails db:create
    $ rails db:migrate

And then:

    $ rails s

This project uses:

- rails '~> 5.2.0'
- sqlite3
- rails-i18n
- httparty
- nokogiri
- kaminari
